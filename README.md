# بک‌اند داروخانهٔ دکتر پیرصالحی

این سرور، رابط بین مینی‌اپ کاربر و پنل داروساز است.

## فایل‌ها چه کاری می‌کنند؟

- **server.js** — همهٔ مسیرهای API اینجاست (ثبت سفارش، دیدن صف، قیمت‌گذاری و…)
- **db.js** — ذخیره و خواندن اطلاعات از یک فایل JSON ساده (`data/db.json`)
- **package.json** — لیست کتابخانه‌هایی که این پروژه نیاز دارد

## چطور روی کامپیوتر خودم اجرا کنم؟ (فقط برای تست)

```
npm install
npm start
```

بعد آدرس `http://localhost:3000` را در مرورگر باز کن — باید پیام «سرور روشن است ✅» را ببینی.

## این روی اینترنت چطور بالا می‌آید؟

طبق مراحلی که با هم رفتیم: این پوشه را در یک Repository گیت‌هاب می‌گذاریم،
بعد Render.com را به همان Repository وصل می‌کنیم تا خودش این را ۲۴ ساعته اجرا کند.
